Orpheus

This is a prototype of a roguelike-like in which the difficulty of the level changes dinamically based on the heart rate of the player.
This was made as a dissertation project for the master degree in computer science, you can read my dissertation abstract *here* or the whole dissertation (in italian) *here*.

In short, it uses an algorithm running on Python to detect and capture the forehead of the player using the webcam, and from that it will estimate the heart rate in a period of time. Given that information, the heart rate is used in Unity to adjust the difficulty of the generation based heart rate value estimated, dividing the level in very small areas.

You can try the prototype here.
